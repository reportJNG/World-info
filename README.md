Country Info Dashboard

A responsive React/Next.js dashboard that displays country information using the REST Countries API
. Supports light and dark themes with a clean, scrollable layout and modern UI components.

Features

Display country flag, name, capital, region, population, languages, currencies, timezones, and Google Maps link.

Responsive layout: Sidebar + Center content.

Scrollable center section and sidebar.

Light/Dark theme support using CSS variables.

Modern, styled scrollbars and buttons.

Fully TypeScript-safe.
Usage

Clone the repository:

git clone <repo-url>


Install dependencies:

npm install
# or
yarn install


Run the development server:

npm run dev
# or
yarn dev


Open http://localhost:3000
 to view the dashboard.
